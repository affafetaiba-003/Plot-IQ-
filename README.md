# PlotIQ — Smart City Property Registry

A full-stack web application for registering and managing 
properties on an interactive map.

## Features

- Interactive Leaflet.js map with real coordinates
- Draw property boundaries on the map
- Live Marla area calculation using Shoelace formula
- House structure editor with drag-to-draw canvas
- Room labeling directly on the map
- Property verification system
- Colour-coded availability polygons
- Search and filter saved properties
- SQL Server database with full CRUD operations
- Node.js REST API backend

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML, CSS, JavaScript |
| Map | Leaflet.js + OpenStreetMap |
| Backend | Node.js + Express.js |
| Database | Microsoft SQL Server |
| API | REST API with 6 endpoints |
